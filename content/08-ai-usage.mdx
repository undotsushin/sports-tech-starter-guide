## 7.1 プロンプトエンジニアリング基礎

### 良いプロンプトの4要素

1. **役割設定**: 「あなたは〇〇の専門家です」
2. **具体的指示**: 何をしてほしいかを明確に
3. **制約条件**: 文字数、形式、注意点
4. **出力形式**: どのような形で回答してほしいか

### 悪いプロンプト例

```
資料作って
```

### 良いプロンプト例

```
あなたは経験豊富な営業資料作成の専門家です。
スポーツメディア向けの広告提案資料を作成してください。

【条件】
- クライアント: 大手スポーツ用品メーカー
- 商品: 新作ランニングシューズ
- ターゲット: 20-30代ランニング愛好家
- 予算: 月500万円
- 期間: 3ヶ月キャンペーン

【出力形式】
1. 提案概要（100文字以内）
2. ターゲット分析
3. 配信プラン
4. 期待効果
5. 予算配分
```

## 7.2 業務別活用事例

### 営業・企画業務

**資料作成**:

- 提案書の構成案作成
- キャッチコピーのアイデア出し
- 競合分析の視点整理

**クライアント対応**:

- メールの文面作成
- 課題整理と解決案の検討
- 会議の議事録作成

### 制作業務

**コンテンツ企画**:

- 記事のタイトル案
- SNS投稿のアイデア
- 動画企画の構成案

**品質向上**:

- 文章の校正・推敲
- SEO対策のキーワード調査
- ユーザーテストの設計

## 7.3 継続学習の方法

### 日常的な習慣

- **毎日5分**: 新しいプロンプトを試す
- **週1回**: 成功事例を記録・共有
- **月1回**: 業務効率化を振り返り

### 学習リソース

- **ChatGPT公式**: 最新機能の確認
- **Prompt Engineering Guide**: 体系的な学習
- **社内Slack**: 成功事例の共有

## 理解度チェック問題 7-1

1. 良いプロンプトの4要素を全て答えてください
2. 営業資料作成時にAIに指示すべき制約条件を3つ挙げてください
3. 継続学習で推奨される「毎日の習慣」は？
4. プロンプトで「あなたは〇〇の専門家です」は何の要素？

**解答**: 1. 役割設定・具体的指示・制約条件・出力形式 2. 文字数・ターゲット・予算（など） 3. 新しいプロンプトを5分試す 4. 役割設定
